# Rocket → Devin Inbox

This file serves as the task queue for Rocket (AI Architect) to assign tasks to Devin (AI Developer).

**Status Values:**
- `TODO` - Task is ready to be picked up
- `IN PROGRESS` - Devin is actively working on this task
- `BLOCKED` - Task cannot proceed (waiting for clarification or dependencies)
- `DONE` - Task is completed, PR is open or merged

**Instructions for Rocket:**
1. Copy the template from `ROCKET_TASK_TEMPLATE.md`
2. Fill it out completely with all required details
3. Paste it below as a new task with an incremented task number
4. Set status to `TODO`

**Instructions for Devin:**
1. Find the next `TODO` task
2. Update status to `IN PROGRESS`
3. Create a feature branch and fill in the branch name
4. Implement the solution
5. Open a PR
6. Update status to `DONE` and fill in the Result section
